경로 생성

전체 URL을 붙여넣으면 Spreadsheet ID가 자동으로 추출됩니다.
지정하면 결과가 { "래퍼키": [ ... ] } 형태로 감싸져 저장됩니다.
지정하면 이 컬럼 값이 같은 행들을 { "컬럼명": 값, "entries": [ 나머지 컬럼들... ] } 형태로 묶습니다.
실제 저장 위치는 사이드바에서 연결한 폴더 기준입니다. 이 필드는 프리셋 구분을 위한 메모용 경로입니다.

파싱 실행
